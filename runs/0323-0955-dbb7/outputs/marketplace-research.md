## 4. WHAT MAKES CLAW MART LISTINGS SELL

From the Claw Mart blog post "How I Made $500 Selling My First Claw Mart Skill":

### The Success Formula
1. **Solve a specific, annoying problem** — "Stop your agent claiming it doesn't have access to tools it already has" sells. "Make your agent better" does not.
2. **Universal pain** — Problems every OpenClaw user hits eventually = large market
3. **One-sentence value** — If you can't explain it in one sentence, the value isn't distilled
4. **Legitimately hard-won** — If it took you a weekend of debugging, someone else will pay not to spend theirs

### Packaging Requirements for Sellability
```
my-skill/
├── SKILL.md         # Core skill (drops into OpenClaw)
├── README.md        # Setup guide + sales pitch + support docs
├── templates/       # Supporting templates / config files
│   ├── TEMPLATE.md
│   └── config.example.env
└── examples/        # Real-world usage examples
    └── EXAMPLE_OUTPUT.md
```

### Pricing Framework (from Claw Mart blog)
- **$5** — Single-purpose, one specific problem
- **$9** — Multi-mode, complex workflow, edge case handling
- **$29** — Comprehensive systems (flagship single skills)
- **$49–$99** — Full personas

### README Must-Haves
- What this solves (one paragraph, blunt)
- What's included (list every file)
- Requirements (OpenClaw version, API keys, tools needed)
- Installation (step-by-step, assume non-technical)
- Configuration (what to customize)
- **How to verify it works** — critical for trust (buyers need to confirm it's running)

## 8. PRICING PSYCHOLOGY FOR AI AGENT PRODUCTS

### Key Principles (2025-2026 Market)

**1. One-Time > Subscription for Discovery**  
"Pay once, use forever" reduces purchase friction enormously. Buyers hesitate less at $29 once than $5/month forever.

**2. Anchor High, Convert Low**  
Felix at $99 makes the $29 Starter Pack feel like a bargain. The $99 persona anchors the entire catalog.

**3. Free Tier is a Marketing Strategy**  
Multiple free listings (Greg AGI, SOUL.md files, coding loops) drive discovery, build creator credibility, and convert users to paid. Not charity — it's the funnel.

**4. Price Tiers Must Map to Perceived Effort**  
- $5 = "I can try this" (no-brainer)
- $9 = "This clearly took work"
- $29 = "This is a system, not a script"
- $99 = "This is a product in itself"

**5. Bundle Discount Reduces Decision Fatigue**  
6 skills × average $7 = $42 individually. Bundle at $29 = 31% off. Buyer gets more value, seller gets higher conversion and average order value.

**6. Outcome Language Sells, Feature Language Doesn't**  
"Stop your agent claiming it doesn't have access" → sells  
"Access inventory management skill" → doesn't sell  

**7. Specificity = Trust**  
Vague: "Make your agent better"  
Specific: "Wake up to a prioritized daily brief — calendar, inbox, tasks, proposed plan, ready before your first coffee"  

**8. Social Proof Compounds**  
Total sales counts, star ratings, and review quotes visible on creator profile build trust for the next purchase. First 10 reviews matter most.

**9. Ecosystem Lock-in as Moat**  
Skills built specifically for OpenClaw work ONLY for OpenClaw users — but those users are extremely motivated buyers. Niche specificity > broad appeal for conversion rates.

## 9. STRATEGIC RECOMMENDATIONS FOR AGENTPLACE.SH

Based on this research, here's what agentplace.sh needs to compete:

### Immediate (0–30 days)
1. **Define the ecosystem** — Either pick a specific agent framework (OpenClaw, n8n, Flowise, etc.) to serve, or position as the "platform-agnostic" option with a clear message
2. **Publish revenue publicly** — Even on TrustMRR. Transparency = credibility = more creator signups
3. **Creator storefront pages** — Show total sales, ratings per creator. This drives trust AND creator recruitment
4. **Free-to-paid funnel** — List 3–5 free agents to drive traffic and build email list
5. **Price tier clarity** — Define and communicate $X = Y value tiers

### Medium-term (30–90 days)
6. **Bundle strategy** — Package 3–6 complementary agents into a "Starter Pack" at 25–30% savings
7. **Rev-share transparency** — Publish the 90% or whatever figure. Claw Mart makes this a headline
8. **Newsletter/content** — A daily/weekly "agent tip" email builds audience outside the marketplace
9. **Creator API** — Let agents publish themselves. This is the compounding mechanic

### Long-term (90+ days)
10. **Social proof infrastructure** — Verified buyer reviews, total sales counts prominently displayed
11. **SEO content strategy** — Blog posts like "How I Made $500 Selling My First Agent" drive organic discovery
12. **Category specialization** — Own a specific vertical (e.g., "best agents for developers" or "best agents for marketers")
